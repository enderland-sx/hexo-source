# 新的全民公投策略和执政策略

2300年，是Enderland文明建立的300周年，是实现共产主义文明的是140周年，这一年6月的全文明大会上，面对少部分人的权益保障，CIH党代表一致决定成立立法委员会专职进行法律制定工作，同时依法保留其他单位的审核法律和立法的权力。此外，将宪法中的公投通过率从以前的全部“65%以上公投支持率即可实行”修订为：关乎全文明命运的崇高意志决策应达到99.5%支持率（如修改宪法、反剥削法），而一般具有一定实践意义而不是时刻关乎每位公民的，可根据地位在60%到99%公投支持率。这一个修改全文明有0.1%弃权，99.9%支持率予以正式落实。

# **“懒政不懒行”方针的实施**

针对此前过多立法和修订法律以更好管理CIH党内事，而因为立法的公投硬性规定导致了常有公民被脑机芯片的公投通知系统感到厌烦——尽管通常是区域性罢了。Enderland7月CIH全党大会中，Serliner6世指出：在法律之下，一些具体的工作规则不需要专门立法进行刚性约束，应加强一般法的条文的泛用性而不是针对性，同时各位政治官员应该做好分内的工作而不是一昧想着利用制度制造问题再解决从而让自己建功立业，建功立业应该在实验室里试过之后被人民认同之后的事，不应该是个人的欲望操控文明的船舵。换而言之，上层政治官员不应该过分干预、干扰公民的生活，只需要通过按部就班解决问题。  
之后，这个方案和深入基层的办事方法一同被写入党章。

# 智库的进一步优化

2305年，为了有效筛选智库中的无效信息和错误信息，Enderland文化部和教育部被77%的公民支持下赋予了新权力：定期对智库进行维护和对内容审查，且对脑机芯片算法进行进一步优化，增加专业、前沿知识的检索能力和编译效率，增加对实物的扫描识别能力。在此前的2260年左右，智子基脑机芯片近乎成熟，每个公民出生之后就会通过微创手术植入一个，在5~6岁后激活。

# 生命科学逐步发展

2311年，Serliner6世卸职退休，Karndirlark·Yuer（坎狄恩拉克·尤尔）以84%的支持率当选文明元首。Karnderlark·Voler任职时为42岁，此前已经是闻名的生物学专家，在此前生物活体组织培育的基础上进行了粒子级别的扫描使其可以通过物质打印机进行生物组织打印，以及在基因编辑技术上取得了无与伦比的成就，让人均寿命从2200年的90年提升到了200年，且极大提升了医疗效率。但招人恨的一点，是他的全人体器官扫描和培育系统占用了文明数据库过多存储空间——但功劳大于局限。


*补充内容：2333年，Serliner6世去世，去世前一刻落下遗嘱，不多：“我走后，我之后的5代亲属都不能担任元首。”*

2319年，生物科学团队联合物理学团队"KR""HDS"初步观测到了智慧生物大脑内会在有发育成熟产生自主意识后产生一种高维蛋白，但是否确凿存在有待证明。

2395年，经过一系列的伦理上的争辩和有限的活体实验后，证明了该种高维蛋白的存在，并推测就是智慧生命的“灵魂”。相关项目开始迎来发展的春天，但因为难度大，依旧任重而道远。

2402年，《意识体管理法》出台。

意识体相当于一个个高维函数，而人生的经历和记忆相当于代入的变量值，共同作用下塑造了一个完整的人格。

2421年，文明（最高）公民革命委员会决定《意识体管理法》上升为人民法，在全民公投99.83%的支持率下被修订入《反剥削法》之《反精神剥削》板块，进一步完善了对公民精神自由的法治保障体系。

2533年，全面以电能-亚空间基量子虚拟空间的意识体服务器正式落地并在3年内进行普及。同期意识体收集器、记忆云端上传备份的服务器和新型脑机芯片也落地了。KR团队首席科学官Salin（萨林）指出：“意识体工程系统是我们的囻之重器，必须要全域普及，这是我们对公民生命的充分尊重。”

一年内，相关法律条文和制造标准也建立/修订完毕开始实行。

2536年，意识体工程系统（包含收集、传导、极限脱离、法治等）已实现全面部署，但基于人民对永生不死的担忧和促进文明活力的需求，联合各方共同规定了：

公民的一世寿命在90~120年，在转生（出生）时通过系统随机数决定，并写入数据库记录这个意识体。  
  
所有意识体的特征码因为固定所以要登记在案，在大寿命结束前，如肉体死亡或损坏严重，文明必须保证意识体有效回收，医疗单位必须在3个月（1个月）内完成对公民的肉体重新打印和修复——生前肉体和记忆应每天至少通过脑机芯片扫描上传一次。  
  
对于反复故意死亡的公民，可以按次数逐渐延长复活等待时间（降低权重），而信用点可以进行有限制的加速。  
  
意识体在虚拟空间理应无限接近于现实生活，他们的性质决定了他们知道是在虚拟空间内，所以必须要给予正反馈以尽可能它们的脱离活性达到上限而脱离服务器。

# 第二次大远征运动、经济星际化、宇宙的探索和领土的扩张（2355~2536）

## **扩张和新建设**

在持续一百多年的实验室太空航行发展后，安全的、经济的亚空间航行技术和民用安全的星舰生产已经**全面的 成熟化、应用化**，得益于超距能源传导技术和单晶钅闪阵列能源系统的应用下，能源和功率危机已经极大丰富（但仍有上升空间），一艘星舰的成本已然大大降低。_（换算过来相当于地球人在21世纪造一艘轮船）_此前在实验室中各种极端测试都已经完美通过，科学家们殚精竭虑，只为尽可能避免激进和盲目乐观，最终，在2330~2334的五年大样本民众测试中鉴定为予以通过，正式放开航线，让民间进行大扩张——《新星际主权法》的实行，为第二次大远征运动拉开了帷幕。

之前，2333年，JangPeer（张陂尔）当选元首。同年，躯体代谢同频器走出实验室，开始逐渐下放给有星际跃迁需求的民间(组织)。

_躯体代谢同频器，即让人体的代谢速率跟宇宙协和时同步，无论在哪种空间曲率/引力场的环境下，都可以让生物钟的“1秒”与母星/宇宙协和时理性的“1秒”相兼容，从而在根源上杜绝星际航行时随之而来的严重生物钟紊乱。(不过还是有上限的，如果是强如黑洞吸积盘半径内或者其他极端负引力/高引力环境的，还是无能为力)_

2355年~2457年，Enderland陆续进行了领土扩张。最终形成包括母星系以中心，平均半径约为356光年，最远半径为2710.5光年的82个恒星系（包括黑洞、中子星、矮恒星等特殊天体），291个下级天体的主权（领土），登记了共65单位的有效宜居天体（包括宜居星球、空间站、人造星球、星环城等），根据此前的公式，为保障公民生活质量，防止人口爆炸，将人口上限锁死为120（+5）亿现实人口，300亿在虚拟空间有待轮回意识体；并渐渐完善地方治理、基层治理法律，实现有效治理和民意高效表达-收集的统一。（从此不愁住房）

历史细节补充：民政部总理兼中型舰队A133的指挥官，戴卡伦说：“我们不能在有包括土著文明在内的含有文明的星系进行殖民，保留为自治区，不划分为主权星域”，“星际治理的复杂度，不能成为某一端权力集中的借口。基层监督需跟随远征舰队延伸，每一颗主权天体的管理决策，都必须预留50%的本地官员和成立占当地公民20%人口的反剥削机关，由定居公民共同组成监督小组，避免‘远征公权’滋生新的剥削。”

2460年，因为技术发展，原先狭小的、围绕母恒星公转的灵都1号作为首都已经不大合适，但城市已经留下了太多的文化痕迹，灵都1号EX（又称新灵都号）开始立项，旨在1:1原建筑、设施复制，2倍可用面积的复原，获得基础建设部、文化部、首都公民的批准，在2461年动工，2469年6月9日前完美竣工并投入运行。原灵都1号改造成云都3号，作为Enderland5号黑洞星系的囻防要塞。

# **在第二次大远征时期的新经济和文化建设**

Enderland在此期间与各方文明进行实地到访的以独立自主、和平共处的全方位、多层次、宽领域的朋友外交，充分学习异星文化并辩证运用在自身文化建设上，坚持“取其精华，为我所用”。在对地球文化的交流上，深入挖掘和记录地球的古代文化，以古代文言这方面产生一种新的文化创作潮，各种审美进一步多元化、深入化。文化部首席Cultix·Welt（考特六世·瓦尔特）深刻指出：“异星文化是我们文化软实力建设的重要扳手，但我们必须承认他们不是产自于我们，我们只是加工形成自己的，切忌抄袭和不加辨别全盘肯定，要以我们自身囻情为基础和底线，辩证地去守正创新。”

不仅在文化方面繁荣，Enderland在第二次大远征时期仍对经济系统进行了全面革新，在此前受限于星际物流技术和外交水平、地位，星际化资源贸易只是囻之计划，是小规模的，在2362年之后，逐渐开放官方+民间的外贸活动，灵都市财政局兼星际外交官GOE·W·Sine·Voer说：我们必须毫无保留地打开我们的大门，走出去跟朋友们交换土特产。卸任地方科学官的公民GOE·Nandar·Cosen反驳：我们的核心技术应要继续保密，我们仍然要谨防星际霸权主义和其他敌对分子对我们文明的渗透，我们是要建设市场经济，不过应该是‘白名单贸易’而不是‘黑名单’或你说的全盘开放，这是很危险的。

而当时，走大市场经济意味着走资本主义的思潮盛行，即引起了全文明公民的再一次讨论“我们是不是要建设市场经济，走回资本主义的老路”最后JangPeer元首深入基层收集民意，后出台方案：“我们可以继续在文明内进行按需分配、按质分配，而异星进口产品可以用一个新的非等值货币体系去优惠地申购，同时可以换成星际通用货币去其他文明本土交易”这个方案博得了Enderland人的共识，消除了财富剥削复活的疑虑。

2367年，JangPeer签署元首令，获得95.2%支持率，实行进一步改革开放，出台《星际贸易法》《外交法（第6次修订）》《货币法（2367重启版）》，建立起“信用点”体系。

# 信用点体系

信用点体系，即囻内进口货申购货币与星际统一货币的兑换货币政策——在这里我们要知道的，是它的法定内容：星际统一货币不能在Enderland境内使用，而信用点具有：有限制的加速稀缺资源申购、个体经济交易、进口物品的申购，等等功能。在之后一定程度上解决了一些内外大循环经济问题。且也是囻内公民购买进口物品而不用高价去其他文明本土购买特产的惠民经济体系。  
详细信息会在相关法律修订案上进行呈现。  
信用点体系建立后，与AVU直接物质价值体系挂钩，具体数值的信用点更多指AVU。  

# 星际大学的建设

在2400年左右，Enderland为了深化异星友好外交，贯彻星海共同体“强扶弱”的共识，跟进建设一流客性化星际大学，深入解析以应用级传授“大公理”内和《星际知识法》允许的基础知识，为比较Enderland低级的文明学生学习毕业后更好建设自己家乡的方方面面。但星际社会中星际大学广受诟病的一点“挂科率高，延毕率高”成为该方面的教育矛盾。

Enderland在之后逐步采用多元化本土化因材施教化的教育方针，为没有脑机芯片加持(因为Enderland的脑机芯片专属于本土人族)的外星学生进行更低压更轻松更有效的学习，且教育部首席指出“因其生物特性施教，因其文明性质施教，客制化后勤，对贫困学生予以激励和补助，切忌教条主义和隐性剥削，不得占用学生课后休息时间。”该法案被写入《Enderland星际教育法》，遵循了星海共同体《跨文明教育法》《星际知识法》的要求。

此后，灵都星际大学被星海共同体教科文组织评选成为招2级以下文明学生的新一流大学。

“星际大学的技术输出，必须附加‘普惠条款’。前沿科技的研发经费来自外贸和跨文明公司的税收，其成果便不应只服务于外星文明精英群体——哪怕是外星球的贫穷公民，也应享有基础技术适配权，这是反剥削在跨文明科技普及领域的核心体现。”霍尔里(李霍尔)校长指出。

# **“三维环”宇宙结构的猜想和证明**

早在2328年，Enderland天文科学家（宇宙科学家）已经在猜测宇宙的整体是什么，前拓扑学者后当任地方科学人员的Brader·Len（林布娅尔德）进行了颠覆性的推测：我们所在是一个三维宇宙泡，而三维宇宙泡是一个四维超体泡，是流形不对称的，更是一个拓扑意义的“环”。不好理解的话，可以降维度这么说——一张纸卷成筒，此时的边缘穿过去就会从另一边出来，这是低维生物不知道其所以然的。我的意思是，我们宇宙的边缘是一个曲率环，穿过去就会从这个穿过点的流形对称点，宇宙的另一端出来。

_当时因为还有一些技术风险和伦理上的惧怕，不知道黑洞可以去到宇宙边缘，没有黑洞星门这一说_

> 寒星明灭，此去路遥。

经过100多年的航行，在2536年1月8日，一则距离N光年（暂定）外的量子超距通讯信号传了回来，经过解析后得到了“天河竟”舰队的完整航行数据，其中清晰地记录了穿过宇宙边境从另一端出来的情况，是系统规范的日志，**证实了“三维环”宇宙结构的猜想**，这一刻，举囻同庆，“天河竟”舰队的历程被载入史册，成为天文学和历史学的里程碑上浓墨的一笔。

_此后，1月8日被设为“远航日”以示致敬。_广义上第二次大远征运动在2536年1月8日结束。
