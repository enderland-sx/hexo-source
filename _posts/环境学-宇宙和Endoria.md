# 环境学 - 第一章 宇宙中的Endoria
## 第一节 Endoira文明所在的宇宙环境

在一些固态星球表面，仰望星空，我们可以看到闪烁的恒星、轮廓模糊的星云，以及相对星空背景有明显位移的行星；有时还可以看到一闪即逝的流星体、拖着长尾的彗星。这些都是宇宙中物质存在的形式，它们连同通过一些空间探测手段才能探测到的星际空间物质，统称为**天体**。

### 天体的定义
- **星云**: 由气体和尘埃组成的呈云雾状外表的天体，主要物质是氢。
- **恒星**: 由炽热气体致密而组成、能散发大量聚变能量的球状或类球状天体。
- **行星**: 在椭圆轨道上绕恒星运行的、近似球状的天体。行星的质量比恒星小，本身不发光，靠反射恒星的光而发亮。
- **卫星**: 环绕行星运转的天体。卫星的大小差别很大。
- **矮行星**: 具有行星级质量，既不是行星，也不是卫星的天体。它直接环绕着恒星，并且自身的重力足以达成流体静力平衡的形状（通常是球体），但未能清除邻近轨道上的其它小天体和物质。

宇宙中的天体都在运动着，运动中的天体相互吸引、相互绕转，形成**天体系统**。天体系统有不同的级别。

### Endoria α所处的天体系统级别
按从低到高的级别，依次为：
**行星系 → 恒星系 → 核星系 → 超星系团 → 宇宙泡**

**具体位置**: 5994152号官方标准宇宙 → 室女座超星系团 → “本”星系群 → 仙女座星系（M31） → **Endoria星系（G型恒星系统） → Endoria-α行星**。 *[数据删除]*

> **备注**: 在宇宙中，我们通常用**光年（ly）** 来计量天体距离。光年是以光速基本常数（c）的速度运动一年的距离，约为 94,605 亿千米。

### Endoria星系内部结构
- **Endoria-α行星系**: 由Endoria-α行星和其卫星“岩-1”组成。岩-1是该行星系中唯一的天然卫星，卫地距离约为53万千米。
- **Endsol恒星系**: 由Endsun（恒星）以及行星、卫星、小行星、彗星、行星际物质等构成。Endsun是该星系的中心天体，其质量占整个星系质量的 99.86%。Endoria-α行星距离Endsun约2亿千米。
- **银河系（核星系）**: 由Endsun和众多恒星（系）以及一个超巨核心天体组成的庞大恒星系统。Endsun与银河系中心的距离约 2.63 万光年。

### 行星动力学
- **行星自转**: 行星绕其自转轴的旋转运动，称为**自转**。Endoria-α的自转轴被称为“地轴”，其磁场南极（地理北极）指向不变。自西向东自转一周的时间为1行星日。
    - **恒星日**: 以星系内恒星为参照，长度为 **20 小时**，是星球真正的自转周期。
    - **远星日**: 以遥远恒星为参照，长度为 21 时 58 分 21 秒。
- **自转速度**:
    - **角速度**: Ω = Δθ / Δt。例如，观测一颗卫星在Δt = 300秒内转过Δθ = 15弧度，则角速度 ω = 0.05 rad/s。
    - **线速度**: V = √(GM/r)，其中 G (引力常数) = 6.67430 × 10^-11 米^3 / 千克·秒^2，M为星球质量，r为轨道半径。
- **黄赤交角**: 行星赤道面与公转轨道面（黄道面）的夹角，在Endoria-α为**±12.5°**。这个夹角是行星产生季节交替现象的关键因素，推动了生命的产生和演化。
